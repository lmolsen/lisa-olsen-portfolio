## Installation Instructions
1. Download the repo
2. Open in VS Code
3. Open a terminal and enter "npm install"
4. In the terminal, enter "npm run dev" 
5. Repeat instructions for https://github.com/lmolsen/lisa-olsen-portfolio-server
